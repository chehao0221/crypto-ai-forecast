# 🚀 Crypto AI Forecast (加密貨幣 AI 自動化分析系統)

這是一個專為加密貨幣市場設計的 AI 自動化分析工具。系統每日會自動掃描 Yahoo Finance 交易量前 300 名的加密貨幣，利用 XGBoost 機器學習模型識別具備短期爆發力的標的，並固定監控市場主流幣種。

## 🌟 核心功能
- **全市場海選**：每日台北時間 08:30 自動執行，從交易量前 300 名中篩選出 5 檔預期報酬最優的標的（排除穩定幣）。
- **主流幣監控**：固定監測 BTC, ETH, SOL, BNB, XRP 之 AI 預估趨勢與壓力支撐位。
- **高頻數據優化**：針對加密貨幣 24/7 交易特性，提供高精度的價格預測與支撐點分析。
- **自動化回測**：具備 `crypto_history.csv` 數據庫，自動進行 5 日循環的預測勝率結算。

## 🛠️ 技術棧
- **語言**: Python 3.10
- **模型**: XGBoost Regressor
- **數據來源**: yfinance (Top 300 Volume Search)
- **自動化**: GitHub Actions (Daily 08:30 AM Asia/Taipei)
